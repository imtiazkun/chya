# chya

Stop motion creation software. Built with CMake and Ninja.

## Requirements

- CMake 3.20+
- Ninja
- C++20-capable compiler (GCC, Clang, MSVC)

## Build

```bash
mkdir -p build && cd build
cmake -G Ninja ..
ninja
```

Binary: `build/chya`

## Run

```bash
./build/chya
```
